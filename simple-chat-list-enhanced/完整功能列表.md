## 🎯 功能详细说明

### 1. 表情功能 🎨

**操作步骤**:
1. 进入聊天页面
2. 点击输入框左侧的表情图标 😊
3. 表情选择面板弹出（96个表情）
4. 点击任意表情，自动发送

**技术实现**:
- 8列 × 12行网格布局
- 表情以32px大号字体显示
- 支持Unicode表情字符

**表情列表**:
```
😀 😃 😄 😁 😆 😅 🤣 😂
🙂 🙃 😉 😊 😇 🥰 😍 🤩
😘 😗 😚 😙 🥲 😋 😛 😜
... (共96个)
```

### 2. 图片功能 🖼️

**操作步骤**:
1. 点击输入框右侧的加号 ➕
2. 更多功能面板弹出
3. 点击"图片"选项
4. 图片选择面板弹出（6张图片）
5. 点击任意图片，自动发送

**技术实现**:
- 3列 × 2行网格布局
- 使用占位符图片（可替换为真实图片）
- 图片消息显示为100×100尺寸

**图片资源**:
- chat_image_1 ~ chat_image_6
- 建议尺寸: 300×300px
- 格式: PNG/JPG

---

### 3. 语音功能 🎤

**操作步骤**:
1. 点击输入框左侧的麦克风图标 🎤
2. 开始录音，显示录音时长
3. 再次点击麦克风图标停止录音
4. 自动发送语音消息

**技术实现**:
- 录音计时器（每秒更新）
- 显示录音时长（如：5"）
- 语音消息显示时长和麦克风图标
- 自己发送的语音显示绿色背景

**注意事项**:
- 当前为模拟录音
- 实际使用需添加录音权限
- 需使用 @kit.AudioKit 实现真实录音

4. 消息变为"消息已撤回"

**技术实现**:

- 添加系统消息提示
- 仅支持撤回自己的消息

**限制条件**:
- 只能撤回自己的消息
- 建议限制在2分钟内

## 📊 数据模型

### messageObj 消息对象

```typescript
class messageObj {
  id?: number;                    // 消息唯一ID
  sender: string;                 // 发送者
  msg: ResourceStr;               // 消息内容
  type?: MessageType;             // 消息类型
  duration?: number;              // 语音时长
  timestamp?: number;             // 时间戳
  status?: MessageStatus;         // 消息状态
  isRecalled?: boolean;           // 是否已撤回
  isRead?: boolean;               // 是否已读
  replyTo?: messageObj;           // 引用回复的消息
}
```

### Item 联系人对象

```typescript
class Item {
  id: number;                     // 唯一标识
  image: Resource;                // 头像
  name: string;                   // 名称
  message: Array<messageObj>;     // 消息列表
  isNewMessage: boolean;          // 是否有新消息
  time: ResourceStr;              // 消息时间
  isTop: boolean;                 // 是否置顶
  unreadCount: number;            // 未读消息数
  lastMessageTime: number;        // 最后消息时间
  isTyping: boolean;              // 对方是否正在输入
  draft: string;                  // 草稿内容
  isGroup: boolean;               // 是否为群聊
  members?: string[];             // 群成员列表
}
```

### MessageType 消息类型枚举

```typescript
enum MessageType {
  TEXT = 'text',       // 文本消息
  EMOJI = 'emoji',     // 表情消息
  IMAGE = 'image',     // 图片消息
  VOICE = 'voice',     // 语音消息
  SYSTEM = 'system'    // 系统消息
}
```

### MessageStatus 消息状态枚举

```typescript
enum MessageStatus {
  SENDING = 'sending',     // 发送中
  SENT = 'sent',           // 已发送
  DELIVERED = 'delivered', // 已送达
  READ = 'read',           // 已读
  FAILED = 'failed'        // 发送失败
}
```

## 📈 性能优化

### 已实现优化

1. **消息分页加载**
   - 每次加载10条消息
   - 避免一次性渲染大量数据

2. **状态管理优化**
   - 使用 `@State` 管理状态
   - 使用 `@Track` 追踪属性变化

3. **列表渲染优化**
   - 使用 `ForEach` 提供唯一key
   - 避免不必要的重新渲染

## 🎓 学习要点

### HarmonyOS 核心技术

1. **组件化开发**
   - @Component 装饰器
   - @Entry 页面入口
   - @Builder 构建函数

2. **状态管理**
   - @State 本地状态
   - @Provide/@Consume 跨组件
   - @StorageProp 全局状态

3. **UI组件**
   - List 列表组件
   - Grid 网格组件
   - Refresh 下拉刷新
   - Navigation 导航组件

4. **手势交互**
   - swipeAction 左滑操作
   - onItemDrag 拖拽排序
   - onClick 点击事件

### 设计模式

1. **观察者模式**
   - @Observed 类观察
   - @Track 属性追踪

2. **状态模式**
   - 消息状态管理
   - UI状态切换

3. **策略模式**
   - 不同消息类型处理
   - 不同操作策略
