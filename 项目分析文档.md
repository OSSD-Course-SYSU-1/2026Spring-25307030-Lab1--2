# HarmonyOS TTS（文本转语音）示例项目详细分析

## 项目概述

本项目是一个基于HarmonyOS的文本转语音（Text-to-Speech, TTS）示例应用，使用ArkTS语言开发。该项目展示了如何使用HarmonyOS基础语音服务提供的文本转语音能力，实现对文本的语音合成及播报功能。

### 项目基本信息
- **项目名称**: core-speech-kit-sample-code-ark-ts-kit-ttsdemo
- **Bundle名称**: com.huawei.hms.ttsdemo
- **版本**: 1.0.0 (1000000)
- **开发语言**: ArkTS
- **UI框架**: ArkUI
- **HarmonyOS SDK版本**: 5.0.0(12)
- **支持的设备**: 手机、平板、2in1设备

##### 2.2.4 核心功能方法

###### 1. createByCallback() - 回调方式创建引擎
```arkts
private createByCallback() {
  // 设置引擎创建参数
  let extraParam: Record<string, Object> = {
    "style": 'interaction-broadcast',
    "locate": 'CN',
    "name": 'EngineName'
  }
  let initParamsInfo: textToSpeech.CreateEngineParams = {
    language: 'zh-CN',    // 语言：中文
    person: 0,            // 音色：0
    online: 1,            // 在线模式：1
    extraParams: extraParam
  };
  try {
    // 调用createEngine方法
    textToSpeech.createEngine(initParamsInfo, (err: BusinessError, textToSpeechEngine: textToSpeech.TextToSpeechEngine) => {
      if (!err) {
        console.log('createEngine is success');
        ttsEngine = textToSpeechEngine;
      } else {
        // 引擎创建失败时，返回错误码1002300005
        console.error("errCode is " + JSON.stringify(err.code));
        console.error("errMessage is " + JSON.stringify(err.message));
      }
    });
  } catch (error) {
    let message = (error as BusinessError).message;
    let code = (error as BusinessError).code;
    console.error(`createEngine failed, error code: ${code}, message: ${message}.`)
  }
}
```
**参数说明**:
- `language`: 语言代码，如'zh-CN'（中文）、'en-US'（英文）
- `person`: 音色索引，0表示默认音色
- `online`: 1表示在线模式，0表示离线模式
- `extraParams`: 额外参数
  - `style`: 交互风格，'interaction-broadcast'表示交互式播报
  - `locate`: 地区，'CN'表示中国
  - `name`: 引擎名称

**错误码**:
- `1002300005`: 引擎创建失败（引擎不存在、资源不存在或创建超时）

###### 3. speak() - 开始语音播报
```arkts
private async speak() {
  // 设置播报相关参数
  let extraParam: Record<string, Object> = {
    "queueMode": 0,          // 队列模式：0表示清空队列
    "speed": 1,              // 语速：1表示正常速度
    "volume": 2,             // 音量：2
    "pitch": 1,              // 音调：1表示正常音调
    "languageContext": 'zh-CN',  // 语言上下文
    "audioType": "pcm",      // 音频类型：PCM格式
    "soundChannel": 1,       // 声道：1表示单声道
    "playType": 1            // 播放类型：1
  }
  let speakParams: textToSpeech.SpeakParams = {
    requestId: UuidBasic.createUUID(),  // 请求ID，使用UUID生成
    extraParams: extraParam
  }
  // 设置回调监听器
  this.setListener();
  // 调用speak方法开始播报
  ttsEngine.speak(this.originalText, speakParams);
}
```
**参数说明**:
- `requestId`: 请求唯一标识符
- `queueMode`: 队列模式
  - 0: 清空当前队列，立即播放
  - 1: 追加到队列末尾
- `speed`: 语速，范围0-5
- `volume`: 音量，范围0-5
- `pitch`: 音调，范围0-5
- `audioType`: 音频类型，支持"pcm"、"mp3"等
- `soundChannel`: 声道数，1表示单声道，2表示立体声

**错误码**:
- `1002300007`: 合成和播放失败（引擎未初始化）
- `1002300006`: 服务忙碌（连续调用speak）

###### 4. setListener() - 设置播报监听器
```arkts
private setListener() {
  let speakListener: textToSpeech.SpeakListener = {
    // 开始播报回调
    onStart(utteranceId: string, response: textToSpeech.StartResponse) {
      console.log('speakListener onStart: ' + ' utteranceId: ' + utteranceId + ' response: ' + JSON.stringify(response));
    },
    // 完成播报回调
    onComplete(utteranceId: string, response: textToSpeech.CompleteResponse) {
      console.log('speakListener onComplete: ' + ' utteranceId: ' + utteranceId + ' response: ' + JSON.stringify(response));
    },
    // 停止播放完成回调
    onStop(utteranceId: string, response: textToSpeech.StopResponse) {
      console.log('speakListener onStop: ' + ' utteranceId: ' + utteranceId + ' response: ' + JSON.stringify(response));
    },
    // 返回音频流回调
    onData(utteranceId: string, audio: ArrayBuffer, response: textToSpeech.SynthesisResponse) {
      console.log('speakListener onData: ' + ' utteranceId: ' + utteranceId + ' sequence: ' + JSON.stringify(response) + ' audio: ' + audio);
    },
    // 错误回调
    onError(utteranceId: string, errorCode: number, errorMessage: string) {
      console.error('speakListener onError: ' + ' utteranceId: ' + utteranceId + ' errorCode: ' + errorCode + ' errorMessage: ' + errorMessage);
    }
  };
  ttsEngine.setListener(speakListener);
}
```
**回调说明**:
- `onStart`: 播报开始时触发
- `onComplete`: 播报完成时触发
- `onStop`: 调用stop方法停止播报完成时触发
- `onData`: 返回合成的音频流数据（ArrayBuffer格式）
- `onError`: 播报过程中发生错误时触发

```arkts
const baseChar: string[] = (() => {

  }
  // 添加小写字母a-z
  for (let i = 0; i < 26; i++) {
    array.push(String.fromCharCode('a'.charCodeAt(0) + i));
  }
  // 添加大写字母A-Z
  for (let i = 0; i < 26; i++) {
    array.push(String.fromCharCode('A'.charCodeAt(0) + i));
  }
  return array;
})();

const UuidObject: Record<string, { prefix: string; count: number }> = {};

export class UuidBasic {
  // 数字进制转换
  private static scaleTransition(value: number, base = baseChar): string {
    if (value < 0) {
      throw new Error('scaleTransition Error, value < 0');
    }
    if (value === 0) {
      return base[0];
    }
    const radix = base.length;
    let result = '';
    let resValue = value;
    while (resValue > 0) {
      result = base[resValue % radix] + result;
      resValue = Math.floor(resValue / radix);
    }
    return result;
  }

  // 生成UUID
  public static createUUID(prefix = '') {
    let uuidObject = UuidObject[prefix];
    if (uuidObject === undefined) {
      const str = `${this.scaleTransition(Date.now())}-${this.scaleTransition(
        Math.floor(Math.random() * 10000000000),
      )}-`;
      if (prefix) {
        uuidObject = {
          prefix: `${prefix}-${str}`,
          count: 0,
        };
      } else {
        uuidObject = {
          prefix: str,
          count: 0,
        };
      }
      UuidObject[prefix] = uuidObject;
    }
    return uuidObject.prefix + this.scaleTransition(uuidObject.count++);
  }
}
```
**功能**:
- 基于时间戳和随机数生成唯一ID
- 支持前缀参数，便于区分不同类型的请求
- 使用62进制（0-9、a-z、A-Z）编码
- 确保本地环境唯一性

## 七、使用说明

### 7.1 运行环境要求
- **系统版本**: HarmonyOS 5.0.0 Release及以上
- **DevEco Studio版本**: DevEco Studio 6.0.0 Release及以上
- **HarmonyOS SDK版本**: HarmonyOS 6.0.0 Release SDK及以上
- **支持设备**: 华为手机、华为平板、2in1设备

### 7.2 运行步骤
1. 使用DevEco Studio打开项目
2. 连接HarmonyOS设备或启动模拟器
3. 点击运行按钮，将应用安装到设备
4. 在设备桌面找到"ttsDemo"应用图标并启动

### 7.3 功能操作
1. **初始化引擎**: 点击"CreateEngine"或"CreateEngineByPromise"按钮创建TTS引擎
2. **开始播报**: 点击"speak"按钮，播报默认文本
3. **修改文本**: 在文本框中输入或修改要播报的文本
4. **查询音色**: 点击"listVoicesCallback"或"listVoicesPromise"查询可用音色
5. **停止播报**: 点击"stop"按钮停止当前播报
6. **检查状态**: 点击"isBusy"按钮检查引擎是否忙碌
7. **关闭引擎**: 点击"shutdown"按钮关闭引擎并释放资源

### 7.4 测试错误场景
- **测试不支持的语言**: 点击"createOfErrorLanguage"按钮
- **测试不支持的音色**: 点击"createOfErrorPerson"按钮
- **测试非法文本**: 点击"SpeakIllegalText"按钮

## 九、注意事项

1. **引擎创建**: 在调用speak之前必须先调用createEngine创建引擎
2. **资源释放**: 使用完毕后应调用shutdown释放引擎资源
3. **并发控制**: 连续调用speak需要等待前一次播报完成，否则会返回1002300006错误
4. **参数验证**: 确保language、person等参数值正确，否则会返回相应错误码
5. **文本长度**: 输入文本长度不能超过限制，否则返回1002300001错误
6. **网络要求**: 在线模式需要网络连接
